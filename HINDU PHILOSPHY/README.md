#  Hindu Philosophy
Dharma AI based on hindu philosophy(geeta,veds and upanishad)

## Files included
index.html
style.css 
script.js 

## Feature
Displays selected verses from the Bhagavad Gita
Provides clear and simple meanings for better understanding
Explains how each verse can be applied in real life situations
Focuses on practical, relatable insights instead of robotic explanations
Aims to connect timeless wisdom with everyday life

## Purpose

This project is built to practice and understand JavaScript concepts such as prompts, async/await, and error handling using try-catch.
